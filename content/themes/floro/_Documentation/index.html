What is diferent in this template?

Floro is based on Gridalicius grid system.

The responsive system, is divided on 3 levels, from smartphones, normal desktops and big desktops (2000px and more).

Image post, are really special, with special CSS3 effects in order to stand out images.

The menu is configured like RSS reader. The menu reads automatically the latest posts from your own blog.

The menu and content, are animated sequentially with only CSS3.

Installation

Local Installation

1 - Copy Floro theme on folder YourGhostDir/content/themes/

2 - You have to restart Ghost in order to refresh the theme list.

3 - Go to settings Ghost admin/settings and set "theme" = "floro" and save it to activate.

4 - Reload your web to see your new theme.

Installation for Ghost.org users

1 - For starting, you have to create an account on Ghost.org, just clicking on "Sign in", or if you have an account, just login.

2 - When you are log in, click on "My blogs".

3 - To create a new blog, click on "New Blog", fill the data and click on "Create New Blog".

4 - To apply Floro template to a blog, go to "My blogs" and click on "Edit" blog's button.

5 - Go to "Custom Theme" input, you can upload the template there. Template is located on folder _Documentation/floro[INSTALL FOR GHOST.ORG].zip.

NOTE: If your Ghost is located on any other platform, maybe this installation does not work for you. You should read documentation from your own hosting services.

Select your color

You can select your favorite color. Go to demo and select it. Copy the color code.

To apply your fav color, go to "floro/assets/css/main.css" and go to last block, "19.Color"

All colors are separated from the other styles. Just only change it here and all is ready.


                        /*Font color*/
                        h1 *,h2 *,h3 *,h4 *,h5 *,h6 *,h1,h2,h3,h4,h5,h6, ul > li:before, ul > li > ul > li:before, ol > li:before, #menu a[class*="fa-"],a, a:hover,#menu a[class*="fa-"]:hover
                        {color:#39CFAA; }

                        /*Backgroud selection color*/
                        ::-moz-selection, ::-webkit-selection, ::selection
                        {background:#39CFAA ;}

                        /*Background color*/
                        .button:hover, .post.featured:before, .post-footer a[class*='fa-']:hover, #menu
                        {background:#39CFAA ;}

                        /*Borders*/
                        blockquote{ border-left: 2px solid #39CFAA; border-right: 2px solid #39CFAA; }
                        input:focus, textarea:focus,select:focus{ border:1px solid #39CFAA;}
                        #disqus_thread, .post{ border-bottom: 15px solid #39CFAA; }

NOTE: The default color are set by default on #39CFAA

Gridalicius

Floro use Gridalicius to create columns. This columns are set width=450px and gutter=30px.

You can to change this values on file "floro/assets/js/main.js", "2.Gridalicious" block.

                        var MyGridWidth = 450;
                        var MyGridMargin = 30;

NOTE: Here is a full documentation of Gridalicius.

NOTE: Gridalicius is disabled for Smarthphones to improve performance.

Custom Nice Scrool

We use Jquery Custom scrool to get the preset scrooll nicer.

If you want to change colors, go to floro/assets/js/main.js on block "1.Nice Scrooll"

                        var MyNSFrontColor = "#333333";
                        var MyNSBackgroundColor = '#989898';

NOTE: Here is a full documentation of NiceScroll.

NOTE: This plugin is disable for Smarthphones to improve performance.

Google Analytics

Floro is ready for Google Analitycs

1. First, you must to register on Google Analitycs HERE

2. Enter your blog url, like http://www.myblog.com

3. Google Analitycs creates an ID for your web, it looks like this UA-111111-1. Copy this ID.

4. Go to "main.js" file, located on "floro/assets/js/main.js" and go to "9. Google Analytics" block.

5. Paste here your Analitycs code and your URL. The code looks like this:

                        var MyGoogleAnalyticsID = 'UA-111111-1';
                        var MyBlogURL = 'myblog.com'

RSS

Menu loads a quik post links from RSS.

The URL from RSS is auto-detected only when the Web is on final server.

To customize it, open file "floro/assets/js/main.js" and go to "4.RSS Loader" block.

You can change url by your RSS and number of post to load.

                        var MyRSSURL = 'http://myblog.com/rss/';
                        var MyPoststoLoad = '6';

NOTE: The URL from RSS is auto-detected only when the Web is on final server. Meanwhile, you can try it using a manual configuration.
